\section{Vintage Mode}
Most basic vim commands should work (command mode only obviously) and can be preceded with a count. \\
\keys{.} Repeat \\
\keys{d}, \keys{count} (optional), \keys{motion}/\keys{dd}/\keys{D} Delete motion (e.g. a word) or delete line or delete line to end  \\
\keys{c}, \keys{count} (optional), \keys{motion}/\keys{C} Change motion or change until end of line (enters insert mode) \\
\keys{y}, \keys{count} (optional), \keys{motion}/\keys{yy} Yank motion or yank (copy) Line \\
\keys{p}/\keys{P} Paste yanked/deleted text after/before the cursor \\
\keys{0}/\keys{\$} Move to the beginning/end of a line \\
\keys{g} Most of \keys{g}'s power is not implemented in Vintage Mode. You can use the basic combinations \keys{gg} and \keys{G} to move to the top/bottom of the file. \keys{number}, \keys{gg} to move to that line. \keys{ga} to display character encoding in the status bar. \\
\keys{a}/\keys{A} Append to cursor/line (enters insert mode) \\
\keys{i}/\keys{I} Insert before cursor/line (enters insert mode) \\
\keys{w}/\keys{W} Move forward a Word/WORD (WORDs are whitespace-separated words) \\
\keys{b}/\keys{B} Move backwards a Word/WORD \\
\keys{e}/\keys{E} Move to the end of a Word/WORD \\
\keys{f}, \keys{character}/\keys{F}, \keys{character} Find (character) forward/backwards on the current line \\
\keys{r}, \keys{character} Replace character under cursor \\
\keys{t}/\keys{T} 'till (forward/backward) Used in combination with motions like \keys{c} or \keys{d}. For example, \keys{d}, \keys{t}, \keys{\{} deletes everything until the next \texttt{\{} character (on the current line).  \\
\keys{o}/\keys{O} Open line below/above (enters insert mode) \\
\keys{\{}/\keys{\}} Move to the beginning/end of a paragraph \\
\keys{s}/\keys{S} or \keys{cc} Substitute character/line (enters insert mode) \\
\keys{H}(igh)/\keys{M}(id)/\keys{L}(ow) Move cursor to the top/middle/bottom of the screen \\
\keys{J} Join Lines \\
\keys{ZZ} Close File \\
\keys{x} Delete character under cursor \\
\keys{X} Backspace \\
\keys{v}/\keys{V} Enters Visual Mode/Visual Line Mode \\
\keys{n}/\keys{N} Find next/previous occurence of search \\
\keys{>}, \keys{>}/\keys{<}, \keys{<} Indent/Unindent \\
\keys{\textasciitilde} Swap Case \\
\keys{q}, \keys{character}/\keys{@}, \keys{character} Record/play macro

\section{Plug-ins}
\subsection{AdvancedNewFile}
\keys{\cmd+\Alt+N} New file from file/project directory (you can autocomplete directories) \\
\keys{\cmd+\shift+\Alt+N} New file from file/project directory (adds an \texttt{\_\_init\_\_.py} file to each directory created)

\subsection{ColorPicker}
\keys{\cmd+\shift+C} Pops up a color picker and, after picking a color, inserts a hexcode

\subsection{Emmet}
\textbf{remapped} \keys{\ctrl+X} Expand abbreviation (e.g. \texttt{ul>li.listitem*6>a}) \\
\keys{p}, \keys{2}, \keys{0}, \keys{\tab} expands to \texttt{padding: 20px} \\
\keys{f}, \keys{z}, \keys{\tab} expands to \texttt{font-size:}

\subsection{PlainTasks}
Formats your TODO files as checklists. \\
Appending a \keys{:} to words (like so Front-end:) creates headers. \\
\keys{\cmd+\return} Create a new task \\
\keys{\cmd+D} Finish/unfinish a task \\
You can add tags with \keys{@}

\subsection{SFTP}
Most of the explanations are directly copied from \url{http://wbond.net/sublime_packages/sftp/usage} \\

\subsubsection{Remote Workflow}
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+F} Upload the current file \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+C} Upload changed files since last commit \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+N} Upload open files \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+O} Download the remote version of the selected file and overwrite the local copy \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+R} Upload the entire folder the current file is located in \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+E} Download the entire folder the current file is located in \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+I} Download the remote version of a file to a temp location and diff it with the local version \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+Y} Compare file modification times and confirm actions before uploading the selected file or folder. This will also delete files off of the remote server that do not exist locally. \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+D} Compare file modification times and confirm actions before downloading the selected file or folder. This will also delete files off of the local machine that do not exist remotely. \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+B} Compare file modification times and confirm actions before uploading/downloading the selected file or folder. This will never delete files, and will always perform the action that causes the newest file to be present in both locations. \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+M} Watch the file for modifications (from Sublime or other applications). When the file modification time changes, the file will be uploaded. \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+W} Open the server quick panel browser at the remote folder corresponding to the selected file or folder. \\

\subsubsection{Server Workflow}
\keys{\cmd+\ctrl+R}, \keys{\cmd+\ctrl+S} Setup a new server. \\
\keys{\cmd+\ctrl+R}, \keys{\cmd+\ctrl+B} List the servers, with an option to setup a new one. \\
\keys{\cmd+\ctrl+R}, \keys{\cmd+\ctrl+N} Show the current directory listing for the last used server. \\
\keys{\cmd+\ctrl+R}, \keys{\cmd+\ctrl+E} List the servers for the purpose of editing. \\
\keys{\cmd+\ctrl+R}, \keys{\cmd+\ctrl+D} List the servers for the purpose of deletion. \\

\subsubsection{Both Workflows}
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+S} Show the SFTP activity panel \\
\keys{\cmd+\ctrl+U}, \keys{\cmd+\ctrl+X} Stop the current SFTP/FTP server activity (connection, upload or download)
